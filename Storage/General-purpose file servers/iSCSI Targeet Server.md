# Servidor de destino iSCSI
Es un rol de window server que permite acceso a almacenamiento mediante el protocolo iSCSI.

Util para proporcionar acceso a almacenamietno a usuarios que por algun motivo no tienen acceso a las catpetas compartidas de windows que funcionan bajo SMB


## Caracteristicas

Arranque por red (sin disco): Permite implementar rápidamente muchos servidores sin disco, ahorrando hasta un 90 % de espacio mediante discos virtuales de diferenciación.

Desarrollo, pruebas y laboratorios: Convierte un servidor Windows en un dispositivo de almacenamiento en bloque accesible por red, ideal para probar aplicaciones antes de pasar a una SAN.
